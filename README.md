## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git@github.com:Beefs4000/social-network-api.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. Run app using intergrated terminal
   ```sh
   npm run watch
   ```
